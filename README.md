# 🛍️ UseNohl E-commerce

## 📋 Sobre o Projeto
Primeiro projeto front-end desenvolvido após conclusão do curso de HTML, CSS e JavaScript. Um e-commerce completo de roupas e acessórios com funcionalidades interativas e design responsivo.

## ✨ Funcionalidades:
- ✅ Modo escuro/claro automático;
- ✅ Carrinho de compras dinâmico;
- ✅ Sistema de login/cadastro com localStorage;
- ✅ Catálogo de produtos por categorias;
- ✅ Pré-checkout com garantia estendida;
- ✅ Design responsivo para todos dispositivos;

## 🛠️ Tecnologias:
- HTML5;
- CSS3 (Variáveis, Flexbox, Grid);
- JavaScript (puro);
- LocalStorage;
- BoxIcons.

## 🚀 Como Executar:
1. Clone o repositório;
2. Abra o arquivo `index.html` no navegador;
3. Navegue pelo site e teste as funcionalidades.

## 📱 Responsividade
O projeto se adapta perfeitamente a:
- Desktops (1920x1080);
- Tablets (768px);
- Smartphones (375px).

## 🎯 Principais Desafios
- Implementar modo escuro consistente;
- Gerenciar estado do carrinho;
- Criar interface intuitiva.
